# My Store

## Overview

My Store is a responsive and visually appealing e-commerce landing page developed using HTML and CSS. The website showcases various products, provides information about the store, and includes contact details. It features a clean user interface with smooth navigation, interactive product cards, and modern styling to enhance the user experience.

## Features

- Responsive and modern user interface
- Product showcase section
- Smooth navigation using anchor links
- Interactive product cards with hover effects
- About section highlighting featured products
- Contact information section
- Attractive gradient backgrounds
- Simple and user-friendly design

## Technologies Used

- HTML5
- CSS3
